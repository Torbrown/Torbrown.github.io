Challenge 5: Accessibility Concerns

Both challenges two and three pose problems to the visually impaired. I think I could address the issues posed by these components by creating a screen reader-friendly website. Meaning I would use alternate text to describe each of the parts in enough detail to communicate their function or appearance. The issues posed by these components could easily be addressed with alternate text. A simple, minimal effort, solution, that would improve accessibility greatly.
